# **App Name**: ExamIQ

## Core Features:

- Dynamic Testing Hub: A fluid, responsive interface for students to take assessments across subjects like math, logic, and languages with real-time timer tracking.
- Integrity Sentry: Silent background monitoring that captures browser events such as tab switching and focus loss to ensure academic honesty.
- Academic Performance Tool: A sophisticated tool using generative AI to analyze student error patterns and performance metrics to suggest focused learning pathways.
- Data Shield Filter: Security-focused middleware that sanitizes question data on delivery, ensuring correct answers remain strictly server-side during active testing.
- Admin Control Matrix: A centralized dashboard for educators to manage complex question sets, monitor active sessions, and review student progress.
- Precision Scoring Engine: Atomic server-side processing that calculates scores and records results in a single transaction for total data consistency.
- Secure Identity Relay: Hardened authentication system utilizing environment variables and server-side verification to protect sensitive administrative endpoints.

## Style Guidelines:

- A dark, focused aesthetic anchored by a vibrant Royal Purple (#7D61EA) as the primary brand color to signify intelligence and creativity.
- Background: A deep obsidian with the same violet hue (#141217) providing a high-end, glare-free reading environment.
- Accent: An electric Celestial Blue (#6B8CF1) for highlighting interactive components and system alerts.
- Font pairing: 'Space Grotesk' (sans-serif) for technical headlines to evoke a precision feel; 'Inter' (sans-serif) for body text and test questions to ensure objective readability.
- Monoline, geometric icons with subtle glowing states that match the accent color for navigational clarity.
- A focus-first centered container layout for tests, minimizing visual distraction while maintaining dense data visualization in the admin portal.
- Swift, functional transitions (150ms) for interface feedback and smooth layout shifts when cycling through questions.

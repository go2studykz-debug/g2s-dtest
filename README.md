# go2study | Интеллектуальная диагностика НИШ

Современная образовательная платформа, использующая искусственный интеллект для оценки уровня подготовки учеников к поступлению в НЗМ (НИШ).

## О продукте
Продукт решает проблему «слепой» подготовки. Вместо простого тестирования, go2study проводит глубокий аудит знаний:
- **Bilingual**: Полная поддержка казахского и русского языков.
- **AI-Driven**: Анализ результатов через Gemini 1.5 Flash определяет характер ошибок (логика, знания или время).
- **Proctoring**: Встроенная защита от списывания с фиксацией нарушений.
- **CRM для образования**: Панель управления лидами, отслеживание брошенных тестов и статуса консультаций.

## Технологии
- **Framework**: Next.js 15 (App Router)
- **Database**: Firebase Cloud Firestore
- **AI Integration**: Genkit + Gemini
- **UI Components**: ShadCN UI + Tailwind CSS

## Для AI (Gemini/Genkit)
Основная логика анализа находится в `src/ai/flows/admin-ai-result-analysis.ts`. Она использует данные о времени ответа и типе вопроса для построения образовательной траектории.
